# Lu Vieira | Desenvolvimento Web & Marketing Digital

Um portfólio moderno e responsivo, unindo tecnologia, criatividade e estratégias de marketing digital.

## Sobre

Desenvolvido por **Lu Vieira** — Dev, Social Media Designer, Copywriter e especialista em SEO e Growth Marketing.

## Funcionalidades
- Design moderno e responsivo
- Animações com Framer Motion
- Dark/Light Theme
- Seções para projetos de TI e Marketing
- Galeria de materiais
- Formulário de contato
- Download de currículo
- Integração com redes sociais

## Tecnologias
- Next.js
- React
- Tailwind CSS
- Framer Motion
- TypeScript
- React Icons

## Como rodar
```bash
git clone https://github.com/luvieirasantos/Portifolio.git
cd Portifolio
npm install
npm run dev
```

Acesse: [http://localhost:3000](http://localhost:3000)

## Deploy
O site está pronto para deploy no Vercel, Netlify ou qualquer serviço que suporte Next.js.
